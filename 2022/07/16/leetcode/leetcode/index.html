力扣92题

给你单链表的头指针 head 和两个整数 left 和 right ，其中 left <= right 。请你反转从位置 left 到位置 right 的链表节点，返回 反转后的链表 。

示例 1：
输入：head = [1,2,3,4,5], left = 2, right = 4 
输出：[1,4,3,2,5]
示例 2：

输入：head = [5], left = 1, right = 1
输出：[5]
提示：

链表中节点数目为 n
1 <= n <= 500
-500 <= Node.val <= 500
1 <= left <= right <= n


进阶： 你可以使用一趟扫描完成反转吗？

92题解题思路如下

充分利用链表相关特性，以及反转链表，那么自然而然会想到递归。我们不需要自己实现一个栈，我们利用java的函数执行栈来完成相关的操作。
即：

*遍历该栈使用函数递归的方式；
*java函数执行栈出栈的时候，构造一个ListNode;
注意位置。

class Solution {

    public ListNode reverseBetween(ListNode head, int left, int right) {
        ListNode headCurr=head;
       return  recursiveLeftList(null,headCurr,0,left,right);

    }
    private ListNode recursiveLeftList(ListNode parent,ListNode head,int index,int left,int right){
        if (head==null){
            return parent;
        }
        ListNode lastNode=recursiveLeftList(head,head.next,index+1,left,right);
        if (index<right&&index>=left){
            parent.next=head.next;
            head.next=parent;
            parent=lastNode;
        }else{
            if (parent!=null){
                parent.next=lastNode;
            }else{
                parent=lastNode;
            }
        }
        return parent;
    }

}
